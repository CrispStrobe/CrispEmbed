# PAN Super-Resolution

Whole-image super-resolution based on PAN (Pixel Attention Network). Upscales images 2x or 4x with only ~272K parameters.

## Architecture

Single-path design with pixel attention throughout:

```
conv_first(3→nf) → nb× SCPA → trunk_conv → skip
  → nearest 2× → upconv1 → PA → LReLU → HRconv1 → LReLU
  → nearest 2× → upconv2 → PA → LReLU → HRconv2 → LReLU   (if scale=4)
  → conv_last(unf→3) → + bilinear(input)
```

Each **SCPA** (Self-Calibrated Pixel Attention) block has two branches:

1. **A**: conv1_a(1×1) → LReLU → k1(3×3) → LReLU
2. **B**: conv1_b(1×1) → LReLU → **PAConv** (k2 1×1 → sigmoid mask, k3 3×3 × mask, k4 3×3) → LReLU

The branches are concatenated, projected by conv3(1×1), and added back as a residual. Upsampling is nearest-neighbour 2× followed by **PA** (pixel attention: 1×1 conv → sigmoid → elementwise multiply). The final output adds a bilinear-upscaled copy of the input as a global residual.

The per-tile forward runs as a single `ggml_conv_2d` graph (the same path used by esrgan/safmn/nafnet/restormer); `PAN_SR_SCALAR=1` selects a reference scalar implementation for A/B comparison.

## Model

- **Source**: PaddleGAN PAN (Pixel Attention Network)
- **License**: Apache-2.0
- **Parameters**: ~272K (nf=40, unf=24, nb=16, scale=4)
- **GGUF**: [`cstr/text-super-resolution-gguf/pan-x4-f16.gguf`](https://huggingface.co/cstr/text-super-resolution-gguf)
- **Converter**: `models/convert-pan-to-gguf.py`

## Parity

ggml graph and scalar paths vs the self-consistent torch reference: cosine similarity = **0.999997** (residual is pure uint8 output rounding). Reference GGUF: `cstr/text-super-resolution-gguf/pan-ref.gguf`, generated by `tools/dump_pan_reference_from_gguf.py` and gated by `test-pan-diff`.

## C API

```c
#include "crispembed.h"

void * ctx = crispembed_pan_sr_init("pan-x4.gguf", 4);
int scale = crispembed_pan_sr_scale(ctx);  // 4

uint8_t * out = NULL;
int ow, oh;
crispembed_pan_sr_process(ctx, rgb_pixels, w, h, 0, 0, &out, &ow, &oh);
// out is w*scale x h*scale x 3 RGB
crispembed_pan_sr_free_image(out);
crispembed_pan_sr_free(ctx);
```

## CLI

```bash
crispembed --pan-model pan-x4.gguf --pan-sr input.png > upscaled.pgm
```

## Python

```python
from crispembed import CrispPanSr

sr = CrispPanSr("pan-x4.gguf")
print(sr.scale)  # 4
out, ow, oh = sr.process(pixels, width, height)
```

## Rust

```rust
use crispembed::CrispPanSr;

let mut sr = CrispPanSr::new("pan-x4.gguf", 4).unwrap();
let (pixels, w, h) = sr.process(&rgb, width, height).unwrap();
```

## Dart / Flutter

```dart
final sr = CrispPanSr('pan-x4.gguf');
final result = sr.process(pixels, width, height);
print('${result.width}x${result.height}');
sr.dispose();
```

## Server

crispembed-server --pan-model pan-x4.gguf
